First Principles Thinking: Breaking Down Problems

Reason from fundamentals instead of analogy.

First principles thinking: break problems to fundamental truths, then rebuild. Instead of "it's always done this way," ask "what's actually true?"

Beyond Analogy

Reasoning by analogy is efficient but can propagate errors. First principles gets to bedrock truth, enabling breakthrough thinking that analogy misses.

What's fundamentally true? Build from there.

Practicing First Principles

  • Question assumptions: What do we assume is true?
  • Break to basics: What's undeniable?
  • Rebuild from scratch: How would this work?
  • Challenge industry norms: Are they actually necessary?

Question everything. Build from truth.
